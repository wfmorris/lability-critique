# lability-critique
R code to accompany a manuscript entitled "What is adaptive demographic lability and when might we expect to see it?" submitted to American Naturalist in February, 2024

R scripts in this file create the figures in the manuscript.  The names of the scripts correspond to the figure numbers in the text and supplement of the manuscript. The script 
< fit skua productivity functions.R > uses data on skua productivity vs lemming data obtained by digitizing Fig. 2 in Barraquand et al., J. Animal Ecology (2014). The digitized data are found in the file < skua data from Barraquand.csv > .
